# Campus Pulse

A modern, smooth-scrolling campus e-magazine experience built with HTML, CSS, and JavaScript.

## 🌟 Overview

Campus Pulse is an interactive digital magazine that showcases campus life through a sleek, responsive design. Featuring smooth scrolling navigation, student spotlights, article highlights, and event photo carousels, it provides an engaging way to explore university news and stories.

## ✨ Features

- **Smooth Scrolling Navigation**: Vertical sidebar with instant page navigation
- **Interactive Student Spotlights**: Clickable cards featuring campus leaders and their achievements
- **Article Highlights**: Featured stories with modal popups for detailed reading
- **Event Photo Carousel**: Auto-advancing slideshow of campus events
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Glassmorphism effects, gradient backgrounds, and smooth animations
- **Accessibility**: Keyboard navigation and screen reader support

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with CSS Grid, Flexbox, and custom properties
- **JavaScript (ES6+)**: Interactive functionality and DOM manipulation
- **Responsive Design**: Mobile-first approach with media queries

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/campus-pulse.git
   cd campus-pulse
   ```

2. Open `index.html` in your web browser

That's it! The magazine will load and be fully functional.
